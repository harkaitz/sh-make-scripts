MAKE SCRIPTS
============

Some scripts to be run from makefiles.

## Help

runit-h

    Usage: runit-h [-v][-D DESTDIR] ...
    
      -a NAME PROGRAM ARGS... : Create a normal waiting service.
    
    Create and manage services using runit.
    
    Platforms: Linux

systemd-h

    Usage: systemd-h [-v][-D DESTDIR][-d DESC] ...
    
      -a NAME PROGRAM ARGS... : Create a normal waiting service.
    
    Create and manage services using runit.
    
    Platforms: Linux
